### 1. What is Microsoft Teams? (2025)
Microsoft Teams is the #1 collaboration platform in the world. It combines:

- Chat (1:1 and group)
- Video/audio meetings
- File storage & sharing
- Apps, tabs, bots, Power Automate
- Voice calling (Teams Phone)
- Integration with 1000+ apps

Part of Microsoft 365 • Free version available at **teams.microsoft.com**

### 2. Teams vs Channels vs Chats

| Type              | Purpose                                  | Example                              |
|-------------------|------------------------------------------|--------------------------------------|
| **Team**          | Group of people + channels               | Marketing Department, Project Apollo |
| **Channel**       | Topic inside a team                      | #general, #design, #sprints          |
| **Private Channel** | Only selected members                  | #hr-confidential                     |
| **Shared Channel**  | Collaborate with external organizations | #vendor-partnership                  |
| **Chat**          | Direct or small group conversation       | 1:1 with Alice or group of 5         |

### 4. Mastering Chat & Calls

| Feature                  | How to Do It                                           |
|--------------------------|--------------------------------------------------------|
| Start 1:1 or group chat  | Click **New chat** → type names                        |
| Format message           | Click **A** (format) under the box – headings, code, etc. |
| @mention                 | @name @team @channel                                   |
| Urgent message           | Type `!!` → notifies every 2 min for 20 min            |
| Schedule message         | Right-click Send → **Schedule**                        |
| Voice call               | Phone icon (top-right)                                 |
| Video call               | Camera icon                                            |
| Share screen             | Share icon during call                                 |
| Record meeting           | … More actions → **Start recording**                   |
| Live reactions           | Raise hand, thumbs up, heart, etc.                     |

### 7. Files, OneNote, Tabs & Apps
- Every channel has a **Files** tab → connected to SharePoint
- Click **+** at top of channel → add Planner, OneNote, Excel, Power BI, etc.
- Upload files by dragging into message or Files tab
- **Loop components** (2025 default) → live collaborative tables, lists, tasks

### 12. Troubleshooting Common Problems

| Problem                        | Fix                                                      |
|-------------------------------|----------------------------------------------------------|
| Can’t join meeting             | Settings → Privacy → **Clear cache**                      |
| Mic/camera not working         | Settings → Devices → test mic/camera                       |
| No notifications               | Settings → Notifications → turn everything on                |
| App slow/crashing             | Switch to **New Teams** or reinstall                       |
| Hidden channels not visible     | Bottom of team list → **Show hidden channels**            |
| File missing                   | Check Files tab → OneDrive/SharePoint (search there too)   |
