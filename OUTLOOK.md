### 1. What is Outlook? (2025 Overview)
Microsoft Outlook is the world’s most widely used email and personal information manager. It includes:

- Email
- Calendar
- Contacts
- Tasks / To Do
- Notes
- Journal (classic only)

**Available as:**
- Desktop app (Windows & Mac)
- Web version (outlook.office.com / outlook.com)
- Mobile apps (iOS & Android)
- Part of Microsoft 365, Exchange, Office 2021/2024

### 2. Versions of Outlook (2025)

| Version                     | Platform           | Notes                                      |
|-----------------------------|--------------------|--------------------------------------------|
| Classic Outlook (Win)       | Windows            | Full-featured, most extensions             |
| Classic Outlook (Mac)       | macOS              | Slightly different UI                      |
| New Outlook (Windows)       | Windows 10/11      | Modern web-based look – toggle in app      |
| New Outlook (Mac)           | macOS              | Default since 2024                         |

### 3. Getting Started

| Task                        | How to Do It                                          |
|-----------------------------|-------------------------------------------------------|
| Add account                 | File → Add Account → enter email                      |
| Switch to New Outlook       | Toggle switch top-right (Windows)                     |
| Switch back to Classic      | Same toggle → “Reload in Classic Outlook”             |
| Dark Mode                   | Settings → General → Appearance → Dark                |
| Enable Focused Inbox        | View tab → Show Focused Inbox                         |

### 4. Mastering Email

| Feature                     | How to Use                                                         |
|-----------------------------|--------------------------------------------------------------------|
| Send email                  | Ctrl+N (Win) / ⌘+N (Mac) or click **New mail**                     |
| Reply / Reply All / Forward | Ctrl+R Ctrl+Shift+R Ctrl+F                                         |
| Search email                | Top search bar – e.g. `from:boss subject:meeting`                  |
| Quick Steps                 | Automate common actions (move + mark read, etc.)                   |
| Sweep rules                 | Select message → Sweep → create rule                               |
| Pin important emails        | Right-click → Pin                                                  |
| Schedule send               | Arrow next to Send → Schedule send                                 |
| @Mention someone          | Type `@` + name in body → they get notified                        |

### 5. Calendar & Scheduling Like a Pro

| Action                      | Shortcut / Method                                          |
|-----------------------------|------------------------------------------------------------|
| New meeting                 | Ctrl+Shift+Q                                               |
| Daily view                  | Ctrl+Alt+1                                                 |
| Work week                   | Ctrl+Alt+2                                                 |
| Week                        | Ctrl+Alt+3                                                 |
| Month                       | Ctrl+Alt+4                                                 |
| Scheduling Assistant        | Click “Scheduling Assistant” in new meeting window         |

| Add shared calendar         | Add calendar → From address book                           |

**Pro tip:** Drag any email onto the Calendar icon → instantly creates a meeting with the email attached!

### 6. Contacts & People
- New contact: **Ctrl+Shift+C**
- Search works on phone, company, notes, etc.
- Create contact group: Select multiple contacts → right-click → **Create group**

### 7. Tasks & To Do (Microsoft To Do Integration)
- Tasks in classic Outlook sync automatically with Microsoft To Do
- Flagged emails appear in To Do
- Use **My Day** in the To Do app for the best daily planning experience

### 8. Focused Inbox & Rules
- Focused Inbox automatically separates important vs. everything else
- Train it: Right-click message → **Move to Focused** or **Move to Other**
- Create rule: Right-click message → Rules → Create Rule

### 11. New Outlook vs Classic Outlook (2025)

| Feature                     | New Outlook       | Classic Outlook   |
|-----------------------------|-------------------|-------------------|
| Speed                       | Faster startup    | Slower            |
| Look                        | Modern, rounded   | Traditional       |
| Add-ins                     | Limited           | Full support      |
| VBA / Macros                | Not supported     | Fully supported   |
| Offline access              | Limited           | Full              |
| Recommended for power users | No                | Yes               |

**Tip:** You can switch between New and Classic anytime with the toggle in the top-right corner.

### 12. Tips & Tricks (Hidden Gems)
- Drag email → Tasks icon → creates task with attachment
- Type `/` in the search bar (New Outlook) → instant command palette
- @mention a file from OneDrive → attaches as shared link
- Pin important folders to **Favorites** (top of folder list)
- Use **Categories** + **Search Folders** for powerful organization
- Right-click any folder → “Add to Favorites”

### 13. Troubleshooting Common Issues

| Problem                     | Fix                                                                 |
|-----------------------------|---------------------------------------------------------------------|
| Emails not syncing          | File → Account Settings → Repair                                    |
| Calendar invites missing    | Check “Shared Calendars” and refresh (Ctrl+Shift+R)                 |
| Search not working          | File → Options → Search → Rebuild index                             |
| Outlook frozen or slow      | Start in safe mode: Win+R → `outlook.exe /safe`                     |
| Too many/large PST files    | Use Online Archive or move old items to archive                     |
